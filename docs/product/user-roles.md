# Roles de Usuario

- **Mujer protegida:** preserva evidencia y recibe orientación clara.
- **Autoridad electoral:** revisa señales, expedientes y rutas de atención.
- **Persona juzgadora:** consulta evidencia, integridad y contexto documentado.
- **Observación ciudadana:** accede a datos agregados y anonimizados.

Los permisos concretos se definirán en una etapa posterior.
