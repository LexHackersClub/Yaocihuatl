# Entity Extraction Prompt

## Propósito

Placeholder para extracción asistida de entidades relevantes en expedientes.

## Inputs Esperados

- Narrativa o evidencia autorizada.
- Lista de tipos de entidad permitidos.
- Criterios de minimización de datos.

## Outputs Esperados

- Entidades detectadas.
- Evidencia textual mínima.
- Confianza o incertidumbre.

## Restricciones

- No enriquecer datos con fuentes externas no autorizadas.
- No revelar datos sensibles innecesarios.
- No completar información faltante por inferencia.

## Notas de Seguridad

Usar anonimización o seudonimización cuando el flujo lo permita.
