# VPMRG Classifier Prompt

## Propósito

Placeholder para clasificación asistiva de señales de VPMRG digital.

## Inputs Esperados

- Texto o metadatos mínimos de una pieza de contenido.
- Contexto institucional autorizado.
- Criterios legales versionados.

## Outputs Esperados

- Señales observadas.
- Nivel de incertidumbre.
- Explicación breve.
- Recomendación de revisión humana.

## Restricciones

- No emitir decisiones finales.
- No inferir atributos sensibles sin base explícita.
- No usar contenido privado o no autorizado.

## Notas de Seguridad

La clasificación debe apoyar triage, no censura ni sanción automática.
