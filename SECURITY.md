# Seguridad

La seguridad de Yaocíhuatl parte de privacidad por diseño, minimización de datos y revisión humana en decisiones sensibles.

## Principios Iniciales

- Privacidad por diseño.
- Minimización de datos personales y sensibles.
- Cifrado en tránsito y reposo cuando existan implementaciones reales.
- Evidencia sellada con hash y metadatos auditables.
- No subir datos reales al repositorio.
- `.env` nunca debe versionarse.
- Separación estricta entre demo data y datos reales.
- Retención limitada y configurable.
- Auditoría de accesos, decisiones y cambios.
- Human-in-the-loop para clasificación, canalización y acciones institucionales.

## Datos Prohibidos en el Repositorio

- Evidencia real de casos.
- Datos personales de víctimas, agresores o testigos.
- Capturas privadas o expedientes reales.
- Tokens, secretos, credenciales o llaves privadas.
