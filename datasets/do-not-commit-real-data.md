# No Subir Datos Reales

Nunca se deben subir al repositorio:

- Datos reales de víctimas.
- Datos reales de agresores o personas señaladas.
- Cuentas personales, handles, teléfonos, correos o domicilios reales.
- Capturas, audios, videos o evidencia sensible de casos reales.
- Expedientes reales o borradores con información identificable.
- Credenciales, tokens, llaves o secretos.

Este repositorio puede contener únicamente datos demo, sintéticos o anonimizados con autorización y documentación suficiente.
