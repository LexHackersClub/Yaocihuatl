# Contribuir

Yaocíhuatl se encuentra en fase de scaffolding inicial. Toda contribución debe mantener el repositorio claro, modular y seguro.

## Reglas Básicas

- Usar commits claros y descriptivos.
- Preferir PRs pequeños y revisables.
- Documentar cambios relevantes.
- No subir secretos, llaves, tokens ni archivos `.env`.
- No subir datos sensibles ni evidencia real.
- Mantener arquitectura modular.
- Agregar tests cuando exista lógica real.
- Actualizar documentación cuando cambie comportamiento.

## Antes de Implementar

- Confirmar que el cambio pertenece al módulo correcto.
- Documentar supuestos y límites.
- Revisar `AGENTS.md`, `SECURITY.md` y `ARCHITECTURE.md`.
- Evitar dependencias innecesarias durante fases tempranas.
