# Yaocihuatl Frontend Demo

Hackathon demo application for the initial Yaocihuatl interface.

This app is intentionally mock-only. It does not connect to the backend, authenticate users, upload files, generate real hashes, call AI services, scrape platforms, or make legal determinations.

Run from the repository root:

```bash
npm install
npm run dev
```

Routes are implemented with Next.js App Router under `src/app`.
